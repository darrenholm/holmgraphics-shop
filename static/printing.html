File Specifications

For best results, please follow these guidelines when submitting files

Accepted File Formats

PDF (preferred), AI, EPS, JPG, PNG, TIFF

Resolution Requirements

Minimum 300 DPI for print-ready files. Larger formats may use 150 DPI.

Color Mode

CMYK color mode preferred. RGB files will be converted.

Bleed & Margins

Include 0.125" bleed on all sides. Keep important content at least 0.25" from trim edge.
